## Waypoint

- Waypoints are generated based on the nodes defined in the map above.
- The filename is determined by the combination of connected node indices.
- The file naming convention follows the format: `N_N.csv`.

  **Example**:  
  Waypoints from node 20 to node 23 are stored in `20_23.csv`.
